FAQ

Questions studio owners ask.

If yours isn't here, email support@klasly.app — Yudai replies personally.

Is the 30-day trial really free? Do I need a card?

Yes — the trial is fully free and no credit card is required upfront. You only enter payment details if you decide to keep using Klasly after day 30.

Can I import my existing members and schedule?

Yes. CSV import handles members; recurring classes are recreated in minutes. Klasly auto-generates sessions for weeks ahead, so each class is set up only once.

How does retreat booking actually work?

Create a retreat and Klasly generates a dedicated landing page with gallery, daily schedule, map, early-bird pricing, waitlist, group booking, and 3-installment payments. Share one link — guests book end-to-end from there.

For shared studios: can instructors collect their own payments?

Yes — that's Shared Studio Mode. Each instructor connects their own Stripe account, sets their own prices, and payments go straight to them. Your studio fee is deducted automatically. Zero manual transfers.

What's the 0.5% platform fee?

A flat 0.5% on member payments processed through Klasly (separate from Stripe's fee). That's how $19/mo covers every feature with no tiers.

Does it work on mobile?

Yes. Klasly is a PWA — members and instructors install it like a native app, and it works offline for viewing schedules and member info.

Can I cancel anytime?

Yes. Cancel from your dashboard, no friction. Annual ($190/yr) cancellations are refunded pro-rata for unused months.
